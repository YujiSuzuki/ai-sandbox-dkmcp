# secrets ディレクトリ

このディレクトリには、AI アシスタントから見えないようにすべき機密情報が含まれています。

## ファイル

- `jwt-secret.key` - JWT トークン署名用の秘密鍵
- `encryption.key` - ノート内容の暗号化に使用する鍵

## セキュリティデモ

AI サンドボックス環境（DevContainer または cli_claude）で動作している場合：
- このディレクトリは空の tmpfs ボリュームとしてマウントされます
- AI アシスタントはこれらのファイルを読み取れません
- 一方、API コンテナはアクセス可能です

これは、シークレットを保護しながら AI と安全に開発する方法を示しています。

## 本番環境での運用

本番環境では：
- 環境変数やシークレット管理サービスの利用を推奨します
- シークレットを git にコミットしないでください
- 鍵は定期的にローテーションしてください
